    Parameters
    Type Name Description
    List<List<int>> CurrentCycleEdges

    A list of edges (each as a pair of vertex indices) that form the current Hamiltonian cycle or path at this step.
    List<int> AddedEdge

    The edge that was added to the cycle in this step, if any. Represented as a list of two vertex indices. Null if no edge was added in this step.
    List<int> RemovedEdge

    The edge that was removed from the cycle in this step, if any. Represented as a list of two vertex indices. Null if no edge was removed in this step (e.g., the final step that completes the cycle).
